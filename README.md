# YuriDot

Dotfiles for Hyprland/Wayland with pywal support.

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/lonelyobserver0/YuriDot/main/setup.sh | bash
```

or

```bash
wget -qO- https://raw.githubusercontent.com/lonelyobserver0/YuriDot/main/setup.sh | bash
```

## Contents

| Folder | Description |
|--------|-------------|
| `fish/` | Fish shell configuration + custom functions |
| `waybar/` | Top bar for Hyprland |
| `wofi/` | Application launcher |
| `gtk/` | Custom GTK theme |
| `gtk-2.0/` | GTK 2 configuration |
| `gtk-3.0/` | GTK 3 configuration |
| `gtk-4.0/` | GTK 4 configuration |
| `cinnamon/` | Cinnamon theme |

## Dependencies

**Required:**
- `git`
- `sass` - to compile SCSS files
- `pywal` - for dynamic wallpaper-based themes
- `quickshell` - for the Wallpaper Picker app

## Pywal

Waybar and wofi themes use colors generated by pywal. To set them up:

```bash
wal -i /path/to/your/image.jpg
```

This generates `~/.cache/wal/colors.scss` used by the SCSS files.

## Wallpaper Picker

The setup script installs `wallpaper-picker.desktop` to `~/.local/share/applications/`. This makes **Wallpaper Picker** appear as a regular application in your app launcher (wofi, rofi, etc.) and application menus.

**Requirements:**
- `quickshell` - required to run the wallpaper picker
- Wallpapers must be placed in `~/Pictures/Wallpapers/`

When launched, it allows you to select a wallpaper and automatically updates the color theme for:
- **Waybar** - top bar
- **Wofi** - application launcher
- **GTK apps** - via pywal integration

This provides a unified theming experience where all components match your wallpaper colors.

**Alternative:** You can also use the Fish function `setbg <filename>` to change wallpaper directly from the terminal:
```bash
setbg my-wallpaper.jpg
```

## Included Fish Functions

| Function | Description |
|----------|-------------|
| `bgp <command>` | Run a command in background with logging |
| `setbg <wallpaper>` | Change wallpaper + update pywal themes |
| `gpumode <mode>` | Switch GPU mode (Hybrid/Integrated/etc) |
| `chdns` | Switch DNS |
| `killvpn` | Kill VPN process |
| `extract <file>` | Extract archives of various formats |

## Backup

The setup script automatically backs up existing configurations to:
```
~/.config/backup_YYYYMMDD_HHMMSS/
```
